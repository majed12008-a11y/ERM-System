# خطة الاختبار الشاملة للنظام
# Comprehensive System Test Plan — ERM Ethics System

## الهدف
إجراء اختبارات صارمة على جميع عمليات النظام البالغة 130+ نقطة نهاية موزعة على 13 وحدة،
لضمان سلامة الدورة المستندية بالكامل واكتشاف أي أعطال قبل تجميد RC1.

## معايير النجاح
- جميع النقاط تعود بـ HTTP 2xx
- لا توجد أخطاء 500 (Internal Server Error)
- أخطاء 403/401 متوقعة فقط عند اختبار الصلاحيات
- استجابة JSON صالحة مع الحقل `success`
- سير العمل الكامل (Project → Application → Review → Decision) مكتمل بدون أخطاء

## التنفيذ
يتم تنفيذ الاختبارات عبر PowerShell باستخدام Invoke-RestMethod
يتم بدء الخادم على port 3000 واختبار API مباشرة
